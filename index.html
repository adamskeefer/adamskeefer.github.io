Mr Fish File Manager

Windows Download (2024.09.25)

Instructions

Run the executable file and select a folder from your system containing .wav files. Mr Fish reads files slated as follows: "Roll [Number] Scene [Number and optional character] Take [number]"
For example, "Roll 2 scene 4A take 3" is in the valid format. Files are renamed within the folder you select, no files are duplicated or deleted.

Threshold will determine the level of sound recorded. The recommended value is 50, but raise if the person on slate was loud, or lower if the person on slate was quiet.

The Roll field is optional. Fill this field in if all the files in the folder are the same roll and/or the person reading slate did not read the roll. The latter is the more useful application of this feature.
